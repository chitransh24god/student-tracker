📋 CSV Format Instructions

Your CSV file must have exactly 3 columns with the following headers:

Column Description Example
name Full name of the student John Doe
email Unique email address john@example.com
password Initial password (min 6 characters) password123

  • Emails must be unique (duplicates will be skipped)
  • All fields are required
  • Rows with missing data will be skipped
  • You'll see a report of how many were added/skipped
