❓ TROUBLESHOOTING

ISSUE: Port 5000 already in use
FIX:   Kill other Python processes or use port 5001
       Edit app.py, change: app.run(debug=True, port=5001)

ISSUE: Python not found
FIX:   Install from: https://www.python.org
       Check "Add Python to PATH"

ISSUE: Dependencies won't install
FIX:   The START.bat script tries each package individually
       If issues persist, try: pip install Flask Flask-SQLAlchemy

ISSUE: Database errors
FIX:   Delete attendance.db file and restart
