## ⚡ Fastest Path to Deployment (3 Steps)

### Step 1: Copy Files to Your Project
Copy these files to your `d:\Downloads\python_cec` (or wherever your student-tracker is):
- `Procfile`
- `runtime.txt`
- `requirements.txt`
- `.streamlit/config.toml`
- `deploy-railway.bat` (if on Windows)

### Step 2: Run Deployment Script
**Windows Users:**
```powershell
cd "d:\path\to\your\student-tracker"
.\deploy-railway.bat
```

**Mac/Linux Users:**
```bash
cd ~/path/to/your/student-tracker
bash deploy-railway.sh
```

**Manual (if scripts don't work):**
```bash
git add Procfile runtime.txt requirements.txt .streamlit/
git commit -m "Add Railway.app deployment configuration"
git push origin main
```

### Step 3: Deploy on Railway.app
1. Visit https://railway.app
2. Sign up / Log in (can use GitHub)
3. Click **"New Project"**
4. Select **"Deploy from GitHub"**
5. Choose **"chitransh24god/student-tracker"**
6. Click **"Deploy"**
7. Wait 2-3 minutes ⏳
8. Your app is LIVE! 🚀
